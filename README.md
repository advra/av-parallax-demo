# Avalar Parallax Demo

## Usage:
```
cd parallax-avalar
yarn
yarn start
```
Note: `unset NODE_OPTIONS` if running fails with vscode.

## Deploy:

#### Local Deployment 
Seen on: http://localhost:1234
```
cd parallax-avalar
yarn start
```
#### Git Pages Deployment
Seen on: https://advra.github.io/av-parallax-demo/
```
cd parallax-avalar
yarn push-gh-pages 
```
